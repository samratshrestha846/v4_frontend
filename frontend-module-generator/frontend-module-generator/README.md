# Simple CRUD module generator

Copy this project in the same directory where your v4_frontend application sits.

If required, run `npm install`

Check .env for required configuration

Run this project: `npm start`

Hit URL `http://localhost:{PORT_DEFINED_IN_ENV}/generate-module?moduleName=YourModuleName`

# command to create
npm run create -- sales-order
npm run create -- task
